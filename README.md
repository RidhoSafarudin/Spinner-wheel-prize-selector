# Project Spinner-wheel-pemilih-hadiah

## Project info

## Deskripsi
Proyek ini adalah aplikasi spinner pemilih hadiah berbasis **quantity** (kuantitas), yang dirancang untuk mengundi pemenang secara adil berdasarkan jumlah stok hadiah yang tersedia. Proyek ini dibuat dan dikembangkan dengan bantuan **Lovable**.

## How to run

Satu-satunya persyaratan adalah memiliki Node.js & npm yang sudah terinstal
Cara Install jika belum
(https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

ikuti langkah2 berikut jika sudah menginstall Node.js dan npm:

```sh
# Step 1: Clone repository menggunakan URL Git project.
git clone https://github.com/RidhoSafarudin/Spinner-wheel-prize-selector

# Step 2: Masuk ke direktori project.
cd Spinner-wheel-prize-selector

# Step 3: Install dependency yang diperlukan.
npm i

# Step 4: Jalankan development server dengan auto-reload dan preview instan.
npm run dev
```


Project Ini Dibuat Dengan:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
